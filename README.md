## Getting Started
### Prerequisites
* node
  ```sh
  [https://nodejs.org/en/](Node.js)
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/lucianoGG/node-typescript.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run service
   ```sh
   npm run dev
   ```
4. Run browser
   ```sh
   `http://localhost:3333`
   ```
